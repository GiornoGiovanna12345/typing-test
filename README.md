# TYPING TEST 

Check out your typing speed by typing out the sentences in the given amount of time!

## Live Demo 
Check it out here: [typing test](https://giornogiovanna12345.github.io/typing-test/)

## Features
- Timed tests (30s, 1min, 2min and 5min)
- Blind mode - to get the actual practice of typing
- Light mode - Offers a "light" version of the same website
- Displays the "Best" score

## Built With 
- HTML 
- CSS 
- JavaScript

## How to Run 
Open index.html in your browser.
